xform

@rexd/xform： 一个基于 mobx 的表单方案。

特点#

  • 上手简单，搭配 mobx 可以进行快速开发
  • 精确渲染，轻松应对大表单场景
  • 重度依赖 mobx，继承 mobx 的优缺点
    • 表单内核采用 mobx 提供的数据结构，无缝对接 mobx 生态
  • 概念较少，学习成本低

字段限制#

  • 字段名称不能包含 .
  • 字段名称不能为 &

开始使用#

在使用表单之前，我们需要先创建一个 Model 对象，然后将其传递给 Form 组件的 props.model。

import { Form, FormItem, RootModel } from '@rexd/xform';
// step1 创建表单模型
const model = new RootModel({
name: '小河马',
phone: '188-8888-8888',
address: '杭州市余杭区文一西路969号',
});
// step2 使用
export function Basic() {
return (
<Form model={model}>
<FormItem component="input" label="姓名" name="name" required />
<FormItem component="input" label="电话" name="phone" required />
<FormItem component="input" label="地址" name="address" />
</Form>
);
}

如果你使用 mobx 管理应用状态，那么可以在 store 中创建 model 对象：

import { observable, computed, makeObservable } from 'mobx';
import { RootModel } from '@rexd/xform';
class Store {
@observable
model = new RootModel({
name: '小河马',
phone: '188-8888-8888',
address: '杭州市余杭区文一西路969号',
});
@computed
get formValues() {
return model.values;
}
constructor() {
makeObservable(this);
}
}

// 以下变量从其他模块中引入：
//  RootModel, Form, FormItem


const model = new RootModel({
  name: '小河马',
  phone: '',
  address: '杭州市余杭区文一西路969号',
  idType: 'id',
});


function BasicUncontrolled() {
  return (
    <Form
      defaultValue={{
        name: '小河马',
        phone: '',
        address: '杭州市余杭区文一西路969号',
      }}
      onSubmit={(values, model) => console.log('onSubmit:', values)}
      onError={(errors, model) => console.log('onError:', errors)}
      onReset={(model) => console.log('onReset')}
    >
      <FormItem component="input" label="姓名" name="name" required />
      <FormItem component="input" label="电话" name="phone" required />
      <FormItem component="input" label="地址" name="address" />


      <Group mt="8px">
        <Form.Submit />
        <Form.Reset />
      </Group>
    </Form>
  );
}

字段校验#

支持使用 validate={ value => error } 进行简单的校验。

名称
*
内联模式
// 以下变量从其他模块中引入：
//  Form, FormItem


function Validation() {
  return (
    <Form>
      <FormItem
        component="input"
        label="名称"
        name="商品编码"
        required
        componentProps={{ placeholder: '请输入商品编码' }}
        validate={(value) => {
          let error;
          if (!value) {
            error = '该字段必填';
          } else if (value.toLowerCase() !== 'sku1234') {
            error = '可以输入 sku1234 试试';
          }
          return error;
        }}
      />
    </Form>
  );
}

联动#

使用 <Form.Effect watch={watch} effect={effect} /> 后，每当 watch 对应的值发生变化时，effect 都会被调用，你可以由此来创建简单的联动。

effect() 被调用时，可以获取到两个参数：

  • value: any: 新的值
  • detail: { prev: any, model: IModel, next: any } 包含旧的值，新的值，以及当前模型的一个对象

非内联模式
// 以下变量从其他模块中引入：
//  RootModel, Form, FormItem, ValuePreview


const model = new RootModel({
  name: '小河马',
  phone: '',
  address: '杭州市余杭区文一西路969号',
  idType: 'id',
});


function BasicEffect() {
  return (
    <Form defaultValue={{ subsidiary: '北京', shops: ['杭州东新东路店'] }}>
      <Form.Effect
        watch="subsidiary"
        effect={(value, { model }) => {
          model.setValue('shops', []);
        }}
      />
      <FormItem
        component="singleSelect"
        label="子公司 (切换子公司后，门店将被清空)"
        name="subsidiary"
        componentProps={{
          dataSource: '北京，上海，浙江，广州'.split('，'),
        }}
      />
      <FormItem
        component="multiSelect"
        name="shops"
        label="门店"
        componentProps={{
          hasClear: true,
          dataSource: [
            '杭州东新东路店',
            '盒马杭州解百店',
            '杭州临平中都店',
            '杭州亲橙里店',
            '杭州庆春店',
            '杭州下沙银泰店',
            '杭州星光大道店',
            '杭州西溪龙湖店',
            '杭州闸弄口店',
          ],
        }}
      />


      <ValuePreview />
    </Form>
  );
}


render(<BasicEffect />)

    </Form>
  );
}

对表单进行操作#

modelUtils 提供了一些针对 model 的工具方法，例如 批量校验 validateAll，清除错误 clearError 等。

fieldUtils 提供了一些针对 field 的工具，例如 validate 等。 具体 API 详见 xform API.

引入方式： import { modelUtils, fieldUtils } from '@rexd/xform';

非内联模式
// 以下变量从其他模块中引入：
//  RootModel, modelUtils, action, Form, FormItem, ValuePreview


const model = new RootModel({
  name: '小河马',
  phone: '',
  address: '杭州市余杭区文一西路969号',
  idType: 'id',
});


function WithActions() {
  return (
    <div>
      <div style={{ display: 'flex', gap: 8, marginBottom: 12 }}>
        <Button onClick={() => modelUtils.validateAll(model)}>校验全部</Button>
        <Button onClick={() => modelUtils.clearError(model)}>清空错误</Button>
        <Button
          onClick={action(() => {
            model.values = {} ;
            modelUtils.clearError(model);
          })}
        >
          重置表单
        </Button>
      </div>


      <Form model={model}>
        <FormItem component="input" label="姓名" name="name" required />
        <FormItem component="input" label="电话" name="phone" required />
        <FormItem component="input" label="地址" name="address" />
        <ValuePreview />
      </Form>
    </div>
  );
}


render(<WithActions />)
